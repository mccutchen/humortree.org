T  E  X  T  U  R  E  /  a pattern generator

T E X T U R E is a PHP script that creates a GIF or PNG image that can be used as a background pattern.

The backgrounds above were created with T E X T U R E. One is a grayscale image, and the other is a color image. Both are generated using random shades of gray and colors. In this example, the actual images are 50 pixels tall and 50 pixels wide, but T E X T U R E can output any size you would want.
